# Arbitrum Reth

A modular, Rust-native execution client for Arbitrum, built on [reth](https://github.com/paradigmxyz/reth).

Arbitrum Reth is a ground-up Rust implementation of Arbitrum's execution layer. Each component lives in its own crate and integrates with reth's trait system for modularity and extensibility.

**Supported networks:** Arbitrum Sepolia (421614)
